## IDLE

기본으로 제공하는 IDLE 사용.
대화형 모드로 파이썬 실행.

## Syntax Error

Syntax Error는 파이썬의 일반적인 오류. 
코드 작성시 콜론, 괄호, 큰따옴표, 작은따옴표를 포함하여 단어의 철자까지 정확히 사용하는지를 확인.

## basic 문법

파이썬 기초에 관한 기본적인 문법과 사용 방법

## 문자열

특수 문자열(이스케이프 시퀀스) 
* \n - 문자열의 줄을 바꿔줌.
* \t - 문자열에 탭 형식의 들여쓰기를 삽입
* \\ - 문자열에 \(역슬래시) 문자를 표현.
* \" - 문자열에 인용 부호를 표현.

## 수학 계한

파이썬의 별표를 곱하기 기호로 사용.
나누기 기호는 슬래시로 사용.

산술 연산자 목록
* + 더하기
* - 빼기
* * 곱하기
* / 나누기
* // 나누기(정수값)
* % 나머지
